# Travel Dataset TBank

### Датасет с информацией об онлайн-платежах и клиентах Т-Путешествия https://disk.360.yandex.ru/d/HzOIoDxXqSDrvA

Содержит данные о **заказах** (типы, статусы), **демографические характеристики** клиентов (пол, возраст, образование) и их **поведенческие метрики** (первые/последние заказы, каналы коммуникации).
